Запуск программы
################

* Склонировать репозиторий.

.. code-block:: bash
   :linenos:

   git clone https://github.com/McForse/configuration-management-mirea

* Перейти в склонированную папку.

.. code-block:: bash
   :linenos:

   cd configuration-management-mirea

* Запустить файл main.py

.. code-block:: bash
   :linenos:

   python main.py
